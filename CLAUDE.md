# House Haven Realty

## Project Overview
Real estate brokerage platform for House Haven Realty (Nashville, TN).

## Stack
- Frontend: Next.js (React)
- Database: Supabase (PostgreSQL)
- Hosting: Vercel
- Auth: Supabase Auth

## Development
```bash
npm install
npm run dev
```

## Environment Variables
See `.env.example` for required environment variables.

## Key Conventions
- Use TypeScript throughout
- Supabase for all database operations and auth
- Deploy automatically to Vercel on push to main
